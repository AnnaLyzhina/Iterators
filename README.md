# Домашнее задание: Iterators. Generators. Yield

Репозиторий содержит решения домашних заданий по теме **итераторы,
генераторы и `yield`**.

## Содержание

Реализованы 4 задания:

### 1. Плоский итератор

Реализован класс `FlatIterator`, который принимает список списков и
возвращает элементы по одному.

**Файл:** `1.flat_iterator/main.py`

------------------------------------------------------------------------

### 2. Плоский генератор

Реализована функция-генератор `flat_generator`, которая возвращает
элементы вложенных списков.

**Файл:** `2.flat_generator/main.py`

------------------------------------------------------------------------

### 3. Итератор с произвольной вложенностью

Реализован класс `FlatIterator`, который работает с вложенными списками
любой глубины.

**Файл:** `3.flat_iterator_any_level/main.py`

------------------------------------------------------------------------

### 4. Генератор с произвольной вложенностью

Реализована функция-генератор `flat_generator`, которая обрабатывает
списки любой вложенности с использованием `yield from`.

**Файл:** `4.flat_generator_any_level/main.py`

------------------------------------------------------------------------
